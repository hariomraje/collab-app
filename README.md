# CollabApp

A React Native collaborative app with chat and editor features.

## 🚀 Features
- Real-time chat
- Collaborative editor
- Firebase integration

## 🛠️ Setup Instructions

### 1. Clone repo
git clone https://github.com/your-username/collab-app.git

### 2. Install dependencies
npm install

### 3. Start Metro
npx react-native start

### 4. Run Android
npx react-native run-android

### 5. Run iOS (Mac only)
cd ios && pod install && cd ..
npx react-native run-ios

## 📁 Project Structure
- src/screens → UI screens
- src/services → backend logic
- src/navigation → navigation
